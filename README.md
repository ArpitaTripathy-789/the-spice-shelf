# The Spice Shelf 

A modern, responsive frontend web application for a spice and masala brand.  
Built with React, TypeScript, and Tailwind CSS, focusing on clean UI, reusable components, and performance.

---

# Features

- Modern React (Vite + TypeScript)
- Component-based architecture
- Responsive design
- Tailwind CSS for styling
- Clean and scalable folder structure

---

# Tech Stack

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **shadcn/ui**
